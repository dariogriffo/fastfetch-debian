# fastfetch

A neofetch-like tool for fetching system information and displaying it in a
pretty way — written mainly in C, with performance and customizability in mind.

This package follows the Debian archive's file layout (presets under
/usr/share/fastfetch/presets, bash/fish/zsh completions, man page) while
tracking upstream releases.

- Upstream: https://github.com/fastfetch-cli/fastfetch
- Packaging: https://github.com/dariogriffo/fastfetch-debian
- Repository: https://deb.griffo.io
